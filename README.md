# NaveRefatorada

O metodo escolhido pelo nosso grupo foi o Observer, o motivo no qual ele foi escolhido foi por conseguir propor a alteração de objetos com base no comportamente de outro.Com isso conseguimos observar o comportamento da nossa propria Nave e aumentar a velocidade da Nave inimiga, isso foi visto a partir das informações no console.

As classes que foram refatoradas foram : Nave , Inimigo e Jogo. Além da implementação de uma classe teste e das interfaces Observer e Subject ,com os metodos updateMovimentoNave,updateMovimentoRapidoInimigo,Update e Update.E addObserver ,removeObserver,notifyObserver,notifyObserver Respectivamente.
